  游戏玩法：按左右间移动飞机，空格键击落敌机，击落敌机有得分。
  
  随着敌机群被歼灭，更新的敌机群速度将会越来越快。
  
  与敌机发生碰撞，或者敌机达到屏幕最下方，玩家损失一条生命值。
  
  游戏为全屏模式，按esc键退出游戏。
